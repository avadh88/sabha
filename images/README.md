# Brahmavidhya Path Images

Place your Brahmavidhya path images in this folder.

## File naming convention (matches config.json defaults)
- path_25.jpg
- path_26.jpg
- path_27.jpg
- path_28.jpg
- path_29.jpg
- path_30.jpg

## Supported formats
JPG, JPEG, PNG, WEBP

## Changing paths
To use different filenames or add more paths, edit the
`bramvidhya_paths` array in `config.json`:

```json
"bramvidhya_paths": [
  { "id": 25, "label": "બ્રહ્મવિદ્યાનો પાઠ – ૨૫", "image": "images/path_25.jpg" }
]
```

If an image is missing, the gallery will show a placeholder icon.
